## üìä Data Jobs Dashboard 2.0 (Single-Page Focus)

![Data Jobs Dashboard 2.0 Demo](DATA%20JOBS%20DASHBOARD/Images/Dashboard%202.gif)

**Overview**  
An advanced, single-page Power BI dashboard analysing **job demand, skill requirements, and compensation trends** in the global data job market from **2023 to July 2025**.

This version represents a **significant improvement over Version 1.0**, with a stronger emphasis on performance, clarity, and analytical depth.

**Key highlights:**
- ~2 million job postings analysed  
- Skill demand and skill intensity per role  
- Median yearly and hourly salary comparisons  
- Identification of top-paying data roles  
- Country-level job market insights  

**Skills applied:**
- Structured Power Query transformations  
- Star schema data modeling  
- Explicit DAX measures and context-aware calculations  
- Dynamic metric switching using DAX  
- Optimised slicers and cross-filtering  
- Clean, recruiter-friendly dashboard design
